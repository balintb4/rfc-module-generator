Model Creator Guide

Step 1: Upload JSON File
The domain model will be generated from your SAP RFC schema file.
The Mendix marketplace contains the RFC Connector for SAP solutions which is required to use the generated domain model in your Mendix application.
Step 2: Select Entities and Attributes
Please select the entities you need below. You can include and exclude specific attributes by clicking on the associated attribute.
You can also search for entities using the search bar.
Step 3: Generate Model
This generated module is a building block for your Mendix app in combination with the RFC Connector for SAP Solutions.
Download the model and import it into your Mendix project (Right click on the app explorer and select import module package).
Step 4: Download Model
Download the generated domain model to your local machine and integrate it into your Mendix app.
